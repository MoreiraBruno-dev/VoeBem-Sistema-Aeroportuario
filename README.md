# VoeBem - Sistema de Gestão Aeroportuária

Este é um sistema web desenvolvido para o gerenciamento de operações de uma companhia aérea, permitindo o controle de aeronaves, passageiros, funcionários e viagens.

## Tecnologias Utilizadas
* **Back-end:** PHP
* **Banco de Dados:** MySQL
* **Front-end:** HTML, CSS e JavaScript
* **Servidor Local:** XAMPP

## Funcionalidades
* Cadastro e listagem de Aeronaves.
* Gerenciamento de Passageiros e Funcionários.
* Controle de Viagens agendadas.

## Como rodar o projeto localmente
1. Certifique-se de ter o XAMPP instalado.
2. Clone este repositório ou baixe os arquivos.
3. Importe o banco de dados:
   - Abra o MySQL Workbench ou phpMyAdmin.
   - Importe o arquivo localizado em `/database/banco-voebem.sql`.
   - Execute
4. Coloque a pasta `src` dentro do diretório `htdocs` do seu XAMPP.
5. Acesse no navegador: `localhost/src/index.php`.
